Steps:-
1> Go to File -> Import -> Maven -> Existing Maven Project
2> Right Click on Project -> Maven -> Update Project
3> Right Click on Project -> Run As -> Run Configuration
4> In Goals type tomcat7:run
5> Click Run
6> Go to browser, in the URL go to localhost:8080
7> Enter the details and click Submit
